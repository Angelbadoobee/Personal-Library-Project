# Personal Library Analytics Dashboard

**Personal Library Analytics Dashboard** is a web-based tool that helps you explore, visualize, and analyze your personal book collection. Gain insights into reading habits, genre preferences, ratings, and series completion—all in an interactive dashboard.

---

## Tech Stack

- **HTML5 & CSS3** – Responsive layout and styling  
- **JavaScript (ES6)** – Dynamic data handling and interactivity  
- **Chart.js** – Interactive charts for data visualization  
- **PapaParse** – CSV file parsing  
- **SheetJS (XLSX)** – Excel file parsing  

---

## Features

- **Upload Your Data:** Supports `.csv`, `.xlsx`, or `.xls` formats  
- **Collection Overview:** Total books, reading completion, total pages, average rating  
- **Genre Analysis:** Number of books, average rating, pages read per genre  
- **Ratings Insights:** Rating distribution, page count vs rating, format-based rating analysis  
- **Series Deep Dive:** Detects book series, tracks volumes, completion rates, and average ratings  
- **Library View:** Filterable and sortable table by genre, status, and format  
- **Sample Dataset:** Quickly test the dashboard with a preloaded collection  

---

## Usage

1. Clone the repository:  
```bash
git clone https://github.com/your-username/book-collection-dashboard.git
```
2. Open index.html in your browser.

3. Upload your book data in CSV or Excel format, or click Load Sample Collection to test.

4. Navigate between tabs to explore your collection.
